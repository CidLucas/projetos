## 🎯 Resumo

Produto para **empresas conectarem seus agentes de IA à base de arquivos corporativa** usando:

- **Turso** como banco de dados (vetorial + relacional, edge-friendly).
- **Grafo de conhecimento** pra representar entidades e relações dos documentos.
- Exposição via **MCP (Model Context Protocol)** para que agentes de IA consumam o contexto de forma padronizada.

**Proposta:** virar a "camada de memória corporativa" plug-and-play pra qualquer agente de IA que precise entender a base documental da empresa.
